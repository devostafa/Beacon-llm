# Beacon LLM

The engine for Beacon's Large Language Model.

## Dependencies

- PyTorch CUDA (+ vision and ao)
- HuggingFace Transformers
- HuggingFace Datasets
- Parameter-Efficient Fine-Tuning (PEFT)
